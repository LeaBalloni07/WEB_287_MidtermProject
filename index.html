Viscious Cycles

WEB-140 Final Ptoject

Web Design

Viscious Cycles was a project I did for my WEB-140 class. It was the introductory class into HTML and CSS. The Project was for a fictional local bicycle store. Requirements for the class was to create at least 4 internal pages. A fluid layout. Links to various sites including social media pages. The site had to be responsive, and also had to include form and some validation. We were given freedom on color choices and also to add to the site if we feel like we can and want to.

Resume Builder

WEB-115 Final Project

Web Design

Resume Builder was my final project for my Web Markup and Scripting class. A basic form was created using HTML and CSS. Form elements was used to accept user input and assigned to variables. These variable values then made up the content of the new page that was generated when the form button (Create Button) was clicked. The resume could only be generated once the email of user input was validated. The main focus was to get familiar and comfortable writing Javascript code.
